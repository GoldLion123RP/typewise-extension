# TypeWise - Smart Text Expansion Extension

A powerful, free text expansion extension for Chrome and Firefox that syncs with GitHub Gists.

## ✨ Features

- 🚀 **Smart Text Expansion** - Type shortcuts, get full text
- 🔥 **Multiple Triggers** - Keyboard shortcuts, hotkeys, context menu
- ☁️ **GitHub Gist Sync** - Free cloud storage using your GitHub account
- 🔐 **Encrypted Storage** - Your data is encrypted before syncing
- 🌓 **Dark/Light Mode** - Automatic theme switching
- 📦 **Import/Export** - Backup and share your snippets
- 🌍 **Cross-Browser** - Works on Chrome and Firefox
- 💯 **100% Free** - No subscription, no limits

## 🚀 Quick Start

1. Install the extension
2. Create your first snippet
3. (Optional) Connect GitHub for cloud sync
4. Start typing your shortcuts!

## 🔒 Security And Keys

- You can run and publish this extension in local-only mode without any GitHub keys.
- GitHub authentication is only required if you enable GitHub Gist sync.
- Use only a public GitHub OAuth Client ID for sync.
- Never store a GitHub Client Secret in this extension codebase.
